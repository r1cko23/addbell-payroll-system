## Step-by-Step Submission Process

### Step 1: Access NPCRS Portal

1. Go to the NPCRS website: **https://privacy.gov.ph/registration/**
2. Click on **"Login"** or **"NPCRS Portal"**
3. Enter your registered email and password
4. Click **"Sign In"**

### Step 2: Navigate to Data Privacy Manual Submission

Once logged in:

1. Look for **"My Registrations"** or **"My Account"** section
2. Find your PIC (Personal Information Controller) registration
3. Click on your registration to view details
4. Look for **"Data Privacy Manual"** or **"Submit Manual"** option/tab
5. Click on **"Upload"** or **"Submit Manual"** button

### Step 3: Prepare Your Data Privacy Manual File

Before uploading:

1. **Convert to PDF Format:**

   - Open `docs/DATA_PRIVACY_MANUAL.md` in a markdown viewer or convert to PDF
   - You can use online converters like:
     - [Markdown to PDF](https://www.markdowntopdf.com/)
     - [Dillinger](https://dillinger.io/) (export as PDF)
     - Or use a word processor to format and save as PDF

2. **File Requirements:**

   - Format: PDF (preferred) or DOC/DOCX
   - File size: Usually maximum 10MB (check NPCRS requirements)
   - File name: Use a clear name like: `GreenPasture_DataPrivacyManual_2025.pdf`

3. **Ensure Manual is Complete:**
   - All sections filled out
   - Company information included
   - DPO information included
   - Appendices included (if applicable)
   - Signed/approved (if required)

### Step 4: Upload the Manual

1. Click **"Choose File"** or **"Browse"** button
2. Select your Data Privacy Manual PDF file
3. Review the file name and size
4. Click **"Upload"** or **"Submit"**

### Step 5: Complete Submission Form

Fill out any required fields:

- **Manual Version:** 1.0
- **Effective Date:** December 17, 2025
- **Last Updated:** December 17, 2025
- **Status:** Active
- **Remarks/Comments:** (Optional) Add any notes if needed

### Step 6: Submit and Confirm

1. Review all information
2. Click **"Submit"** or **"Confirm Submission"**
3. You should receive a confirmation message
4. Save the submission reference number/ticket number

### Step 7: Upload Privacy Notice (If Required)

Some registrations also require Privacy Notice submission:

1. Navigate to **"Privacy Notice"** section (if available)
2. Convert `docs/PRIVACY_NOTICE.md` to PDF
3. Upload the Privacy Notice PDF
4. Submit

## Important Notes

### File Formatting Tips

When converting to PDF:

1. **Use Professional Formatting:**

   - Clear headings and subheadings
   - Table of contents with page numbers
   - Consistent font and spacing
   - Page numbers on all pages

2. **Include Required Elements:**

   - Company letterhead (optional but recommended)
   - DPO signature page
   - Management approval signature
   - Date stamps

3. **File Size Optimization:**
   - Compress PDF if file is too large
   - Remove unnecessary images
   - Use standard fonts

### Common Issues and Solutions

**Issue:** File too large

- **Solution:** Compress PDF or split into multiple files

**Issue:** Format not accepted

- **Solution:** Convert to PDF using a reliable converter

**Issue:** Portal not accepting upload

- **Solution:** Try different browser, clear cache, or use email submission

**Issue:** Missing information

- **Solution:** Review manual checklist before submission
