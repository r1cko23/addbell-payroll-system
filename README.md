# Green Pasture Payroll System

This repository contains the Next.js payroll and attendance platform for Green Pasture People Management Inc.

## Documentation

All historical guides, migration notes, and setup instructions now live under `docs/`. Start with:

- `docs/README.md` – documentation index
- `docs/guides/` – implementation guides (location locking, bi-monthly payroll, etc.)
- `docs/legacy/CLASSIC_README.md` – the original static-app README for reference

## Getting Started

1. `cp .env.example .env.local` (or ensure the Supabase env vars listed in `docs/setup/SETUP.md` are present)
2. `npm install`
3. `npm run dev`

For deployment steps and environment requirements, see `docs/deployment/DEPLOYMENT_GUIDE.md`.
